# Game build kodları

`game için`

- gmake -j20

`db için`
- cd /root/MRYF_GameSrc/Server/db/src/
- gmake clean
- gmake dep
- gmake -j20

`hem game hem db için`
- cd /root/MRYF_GameSrc/Server/
- gmake all
